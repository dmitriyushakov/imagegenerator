Сборка и установка в Linux
-------------------------------------------
Для сборки приложения вам понадобится Qt 5.

qmake-qt5 ImagineGenerator.pro
make
sudo make install

По умолчанию приложение устанавливается в 
поддиректории /usr/local. Если вы мейнтейнер
пакета и вам нужна другая дирректория
попробуйте так:

qmake-qt5 ImagineGenerator.pro PREFIX=/usr
make
make install INSTALL_ROOT=/tmp/pkg

Сборка в Windows
-------------------------------------------
Соберите приложение. В директории приложения
должно быть две папки:
qttranslations - файлы локализации вашей Qt
translations - файлы локализации приложения.

Использование
-------------------------------------------
Для рендеринга изображений используются 
выражения на языке, аналогичном Java Script
(Qt Script).
В поле "Сценарий" необходимо ввести
определения функций и переменных.
Выберите цветное или серое цвет изображения,
в соответствующих полях введите выражение,
вычисляющее значение от 0 до 255.
Если непрозрачность не задана, рисунок
считается непрозрачным.
"Настройки указателя" влияют на значение
переменных x и y.
"Относительно" - от 0 до 1.
Единице соответствует правый/нижний край.
"Абсолютно" - от 0 до ширины/высоты.
После выбора значений нажмите "Отрисовать".
Нажав на кнопку "Координаты" вы попадете
в диалоговое окно выбора системы координат.

Функции
-------------------------------------------
sqrt(val) - квадратный корень
sin(val) - синус
cos(val) - тангенс
tg(val) - тангенс
ctg(val) - котангенс
pow(a,b) - возведение "a" в степень "b"
ln(val) - натуральный логарифм
log(a,b) - логарифм от "b" по основанию "a"
exp(val) - экспонента
strip(a) - отрезает значения, выходящие за
пределы [0;255]
strip(a,b,c) - отрезает значения "a",
выходящие за пределы [b;c] и помещает
в промежуток [0;255]
dist(x,y) - расстояние, вычисленное от
базисов x,y.

Переменные
-------------------------------------------
x, y - в прямоугольной системе координат.
r, a - в радиальной.
relative.x, relative.y - относительные
координаты в прямоугольной системе координат.
absolute.x, absolute.y - абсолютные
координаты.

Константы
-------------------------------------------
pi - число Пи
e - число Эйлера
